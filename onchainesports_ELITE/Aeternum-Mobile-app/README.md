# Aeternum - Esports Betting Exchange

A decentralized esports betting exchange platform where land and assets are represented as NFTs and priced using bonding curves. Built on Solana blockchain with peer-to-peer markets, minimal fees, and complete transparency.

## 🎮 Overview

Aeternum is a next-generation esports betting platform that leverages blockchain technology and NFTs to create a transparent, fair, and efficient marketplace for esports assets and betting. Players can trade esports properties, participate in peer-to-peer markets, and earn from their esports investments with minimal friction.

## ✨ Key Features

- **NFT-Based Assets**: All land and assets are represented as NFTs for true ownership and transferability
- **Bonding Curve Pricing**: Dynamic pricing mechanism ensures fair valuation and smooth liquidity
- **Peer-to-Peer Markets**: Direct trading between users with minimal intermediaries and low fees
- **Solana Blockchain**: Fast, scalable, and cost-effective blockchain infrastructure
- **Smart Contracts**: Secure Rust-based smart contracts for safe asset management
- **Multi-Platform**: Web admin panel, mobile app, and REST API backend
- **High Transparency**: On-chain transactions and verifiable market data

## 🚀 Getting Started

### Prerequisites

- A smartphone with Android or iOS
- A Solflare wallet account
- Some SOL tokens for transactions

### Installation & Setup

1. **Download the App**
   - Download the app using this link: [https://expo.dev/artifacts/eas/aBh5UsgFMj9jmbCZN4kU17.apk](https://expo.dev/artifacts/eas/aBh5UsgFMj9jmbCZN4kU17.apk)

2. **Open the App**
   - Launch the downloaded app on your mobile device

3. **Connect Your Solflare Wallet**
   - Select the wallet connection option in the app
   - Choose Solflare wallet from the available options
   - Approve the connection request

4. **Verify Your Signature**
   - Confirm the transaction in Solflare by verifying your signature
   - This establishes your Solana wallet connection to the platform

5. **Get Initial Funds (Airdrop)**
   - Request an airdrop to bootstrap your account: [https://faucet.solana.com/](https://faucet.solana.com/)
   - This provides you with initial SOL tokens for transactions

### Using the Platform

Once set up, you can:

- **Buy & Sell Assets**: Purchase and sell esports assets and betting exchange shares
- **Trade**: Participate in peer-to-peer markets with other users
- **Invest**: Earn from your esports investments with transparent pricing
- **Exchange**: Access all betting exchange features with minimal fees
