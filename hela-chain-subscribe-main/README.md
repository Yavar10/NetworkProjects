# HeLA Network Project

## Mandatory Submission Guidelines

### Requirements

#### 1. **Repository Setup**
- Public GitHub repo (or GitLab) with this README.md
- One-paragraph project summary and quick start guide included

#### 2. **Contract Code & Tests**
- Smart contract source code
- Comprehensive test suite

#### 3. **Frontend Code**
- Link to live demo
- Full frontend source code included

#### 4. **Deployment Information**
- Deployment addresses for testnet/mainnet
- 3 example transaction hashes with transaction types:
  - Post
  - Like
  - Tip

#### 5. **Live Demo URL**
- Hosted on Netlify / Vercel or direct IPFS gateway

#### 6. **Project Submission**
- Submit the project to the HeLA Repository: [https://github.com/HelaNetwork/NetworkProjects](https://github.com/HelaNetwork/NetworkProjects)

#### 7. **Optional: NFT Art & Metadata**
- Include IPFS links for NFTs if applicable

#### 8. **Optional: Tokenomics/Whitepaper**
- Include tokenomics or whitepaper documentation if your product/dApps uses in-product/dApps token
